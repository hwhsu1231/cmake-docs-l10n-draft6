codegen¶

  • The Ninja Generators and Makefile Generators now produce a codegen build target. See policy CMP0171. It drives a subset of the build graph sufficient to run custom commands created with add_custom_command()'s new CODEGEN option.

  • The add_custom_command() command gained a CODEGEN option to mark a custom commands outputs as dependencies of a codegen target.
