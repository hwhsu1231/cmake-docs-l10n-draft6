INTERFACE_CXX_MODULE_HEADER_UNIT_SETS¶

在 3.25 版本加入.

备注

Experimental. Gated by CMAKE_EXPERIMENTAL_CXX_MODULE_CMAKE_API

Read-only list of the target's PUBLIC C++ module header sets (i.e. all file sets with the type CXX_MODULE_HEADER_UNITS). Files listed in these C++ module header sets can be installed with install(TARGETS) and exported with install(EXPORT) and export().

C++ module header sets may be defined using the target_sources() command FILE_SET option with type CXX_MODULE_HEADER_UNITS.

See also CXX_MODULE_HEADER_UNIT_SETS.
